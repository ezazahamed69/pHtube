• Discuss the scope of var, let, and const?

    • The var keyword is a legacy keyword that was introduced in JavaScript 1.0. It has global or function scope, and can be updated and re-declared. The let keyword was introduced in JavaScript ES6. It has block scope, and can be updated but not re-declared. The const keyword was also introduced in JavaScript ES6. It has block scope, and cannot be updated or re-declared.

  • Tell us the use cases of null and undefined.

    • Null is used to represent the intentional absence of a value. For example, you might use null to represent a missing value in a database. Undefined is used to represent the unintentional absence of a value. For example, a variable might be undefined if it has not been assigned a value yet.

  • What do you mean by REST API?

    • REST API stands for Representational State Transfer Application Programming Interface. It is an architectural style for designing APIs that uses HTTP requests to access and manipulate resources. REST APIs are widely used to connect different applications and systems, and they are a fundamental building block of the modern web.
